# 학교 디지털 출결 제작기

안녕하세요, Sungblab입니다! 제고등학교 학생들을 위한 QR코드 디지털 출결 개발 이야기를 공유하려고 합니다.
기존 수기 출결의 문제점을 깨닫고 나만의 방법으로 해결 해보고자 만들었습니다.

### 1. 기획

저는 풀스택 으로 개발을 기획했고요. 출결 체크를 하는 방법중에서 지문인식,rfid카드,QR코드 등 다양한 방법이 있지만
저는 그 중에서 싸고 접근성이 좋은 QR코드로 선택했습니다. 네이버 QR코드 로그인에 영감을 받아서 반대로 별도의 QR코드
스캐너의 나만의 고유 QR를 인식 시키면 되지 않을까? 라는 생각을 해보았습니다 사실 이미 많이 시중에 존재하는거죠.

#### 기술 스택 선정

- **Frontend**: Vanilla HTML
- **Backend**: Express.js & Node.js
- **Database**: MongoDB
- **인증/보안**: JWT & bcrypt
- **UI/스타일링**: Tailwind CSS

이렇게 결정 했습니다.
네 이건 또 클로드 3.5 소넷과 함께 진행을 했습니다. 일단 저는 먼저 백엔드 코드를 다 짜버리고 그에 맞춰 프론트 코드를 짜보자고 생각해보았습니다.
클로드와 소통을 하며 1000 몇줄의 코드를 짜보았습니다. 그리고 그냥 일반 HTMl 로 프론트 코드를 짜보았습니다. 사실 react도 고려해보았지만...
리액트를 처음 쓰는 저에겐 너무 어렵더라고요 포기했습니다. 그래서 그냥 일반 HTML 로 짜보았습니다. 그리고 초안이 완성되었습니다.

프로젝트 구조는 이렇게 되는데요 index.html 은 로그인 페이지 이고요 여기서 로그인을 하면
hub.html 로 가게 되는데 여기서 일반 사용자면 바로 qr.html 로드 되고
admin 이나 reader 권한이 있다면 hub 에 머물게되고 메뉴를 선택하게 됩니다.
qr코드는 암호화 되어서 30초마다 자동으로 새로고침 되고요, reader 에서는 말그대로 QR코드를 스캔해주는 페이지 입니다.
