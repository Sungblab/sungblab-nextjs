# 🚀 Sungblab: 코딩도 예술처럼, 블로그도 감성 있게

코딩만 하기엔 인생은 짧고, 내 작업물을 그냥 썩히기엔 너무 아깝습니다. 그래서 만들었습니다.  
**Sungblab**은 Next.js 14의 파워와 TinaCMS의 편리함, 그리고 '약간의' 감성을 버무린 개인 포트폴리오 겸 블로그입니다.

![Uploading Screenshot 2026-02-08 154335.png…]()


[https://sungblab.vercel.app](https://sungblab.vercel.app)

---

## 🛠️ 이거 어떻게 만들었냐구요? (기술 스택)

뭐든 최신이 좋잖아요? 그래서 성능과 엔진 소리가 좋은 녀석들로 골랐습니다.

- **엔진**: Next.js 14 (App Router & Pages Router의 기묘한 공존)
- **뼈대**: React 18 & TypeScript (타입 에러는 내 친구)
- **겉모습**: Tailwind CSS & Styled Components (디자인은 감성입니다)
- **두뇌**: TinaCMS (마크다운 파일이랑 싸우기 싫어서 도입했습니다)
- **데이터**: MongoDB & Gray-matter
- **집**: Vercel (클릭 몇 번에 배포 끝!)

---

## ✨ 그냥 블로그는 아닙니다. 이런 것도 돼요.

- **눈 건강 지킴이**: 다크 모드/라이트 모드 지원. 시력은 소중하니까요.
- **수학도 문제없음**: KaTeX 완벽 지원. 수식 적을 때 스트레스 받지 마세요.
- **수다는 GitHub에서**: Giscus로 소통합니다. 댓글 달면 제 GitHub 알림이 울려요.
- **애니메이션 맛집**: Framer Motion과 Typewriter 효과로 '살아있는' 느낌을 줬습니다.
- **글로벌 인재 (지망)**: 다국어(i18n) 지원. 세계로 뻗어 나갈 준비가 되었습니다.

## 🏗️ 로컬에서 돌려보기

이 프로젝트를 직접 돌려보고 싶으신가요? (왜죠? 제 블로그인데!)  
하지만 환영합니다. 아래 명령어로 시작하세요.

```bash
# 의존성 설치 (npm이 열일할 시간입니다)
npm install

# 개발 서버 가동 (TinaCMS와 함께!)
npm run dev
```
